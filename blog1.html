5 Hal yang Saya Peroleh dari Mengikuti Besut Kode Universitas




Hai semua! Akhirnya setelah pos pertama saya tentang anime Re:Zero, saya kembali menulis pos blog lagi. Tentunya ada alasan tertentu saya menulis pos blog ini selain untuk mengerjakan task yang sudah ditetapkan :p

Kali ini saya akan membahas tentang hal apa saja yang telah saya peroleh dari mengikuti lomba Besut Kode Universitas. Oke, tanpa mengulur-ulur dan membuat terlalu banyak drama, langsung saja saya bahas! :)

5 hal yang saya peroleh dari mengikuti Besut Kode Universitas




  1. Pengetahuan tentang Git dan GitHub



    2. Oke, sebagai calon-calon developer atau bahkan yang sudah menjadi developer cilik, sudah sewajarnya tahu tentang Git dan GitHub. Namun untuk yang masih baru dan baru menyelami dunia yang penuh dengan sintaks dan kode alias dunia programming, tentunya Git dan GitHub masih asing di telinga. Begitulah yang terjadi dengan saya. Sekitar satu bulan lalu, saya masih tidak tahu apa itu repositori, pull request, commit, dan semacamnya. Tapi kalau dilihat sekarang, sepertinya saya sudah sedikit lebih baik karena sudah ‘agak’ tahu kalau pull request itu seperti ini, commit itu apa, dan semacamnya. Tidak sebodoh sebelumnya walaupun sekarang masih bodoh sih :p

  2. Sikap disiplin dan tepat waktu



    3. Sering kali apabila ada pekerjaan, kita suka menundanya (termasuk saya yang seharusnya sudah menulis pos ini dari kemarin, malah baru menulis sekarang). Kalau kalian mengikuti lomba ini, setiap 3 hari akan ada mini-task yang harus dikerjakan. Dengan adanya deadline waktu 3 hari, tentunya kita dilatih untuk menjadi lebih disiplin karena kalau sampai mini-task yang harus dikumpul di hari H belum kita kerjakan, maka bersiaplah kita untuk tereliminasi. Nah, kalian bisa lihat, saya yang suka menunda-nunda pekerjaan saja sudah sedikit bertobat karena ada deadline waktu ini. Mau tak mau, harus mau jadi disiplin dan tepat waktu!

  3. Relasi dan Teman baru



    4. Ini yang paling saya sukai di antara semuanya! Dapat teman baru! Dari ikut lomba ini, relasi saya jadi bertambah. Awalnya cuma tahu orang-orang di kampus saya saja, sekarang jadi tahu orang di luar kampus. Para mentor dan peserta di lomba ini ramah-ramah jadi saya juga tidak segan untuk bertanya ini itu dan berusaha berteman dengan mereka. Terkadang ada task yang saya bingung, saya tanya ke peserta lain dan mereka menjelaskan ke saya tanpa ragu atau berpikir ‘Wah dia ‘kan peserta juga, ga mau bantu ah biar tereliminasi’. Tidak ada yang berpikir negatif, semuanya saling membantu untuk jadi lebih baik.

  4. Kemampuan Bahasa Inggris



    5. Perlu kalian tahu, kemampuan Bahasa Inggris kalian bisa bertambah dari mengikuti lomba ini. Kenapa? Karena ada Mr. John! :D Mr. John ini sebenarnya bisa Bahasa Indonesia juga, namun tentunya akan lebih enak berkomunikasi dengan Bahasa Inggris. Di Gitter dan Slack, para peserta juga bicaranya pakai Bahasa Inggris, jadi mau ga mau harus berusaha memahaminya. Saya yang Bahasa Inggrisnya ga terlalu bisa pun berusaha berbicara pake Bahasa Inggris ke Mr. John pas kumpul tugas. Jadi, bisa dibilang, dari lomba ini, saya jadi mengasah kemampuan Bahasa Inggris saya ketika baca chat di grup dan melakukan pull request dari repositori memakai Bahasa Inggris.

  5. Keberanian dan Pantang Menyerah



    6. Dari lomba ini, saya harus bilang kalau saya jadi lebih berani dan pantang menyerah :p Kenapa? Karena saya jadi berani berkenalan dengan orang baru. Jadi ceritanya, saya ada task yang tidak paham dan saya tidak mau tereliminasi begitu saja tanpa berusaha mengerjakan apapun. Saya melihat chat di grup, ada peserta yang memahami task. Saya melawan rasa takut saya dan coba untuk chat personal menanyakan tentang maksud task yang diberikan. Beruntungnya saya, peserta itu baik dan menjelaskan task yang saya tidak paham. Lalu mengenai pantang menyerah, sebenarnya beberapa task lalu ada yang sulit bagi saya dan saya sudah mau menyerah saja, tetapi saat melihat peserta yang lain masih semangat dan terus berusaha, saya pun tidak jadi menyerah dan berusaha mengerjakan task yang diberikan hingga sampai pada hari ini. Ya, meskipun task utama yang mencari 20 bug masih tidak paham, saya akan tetap berusaha cari informasi bagaimana cara mengerjakannya :D

Jadi, begitulah 5 hal yang telah saya peroleh dari mengikuti lomba ini. Jika tahun depan masih ada lomba Besut Kode lagi, jangan ragu untuk coba ikut lomba ini karena lombanya fun kok! *walaupun kadang bikin stress juga*

Baiklah, sampai jumpa di pos berikutnya ya! –thingdiputra
